Two short pages: how to install the harness and what it needs, and a guided first run that ends with a scored comparison table on your own machine.

tomo-labs runs entirely on your hardware.
The only thing that leaves is the model call every agent under test would have made anyway, forwarded through the trace proxy to the free model configured for the sweep.

Start with [installation](/getting-started/installation/), then follow the [quick start](/getting-started/quick-start/).
